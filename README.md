# tmux-tui

tmux-tui is a simple tui for tmux

it enables you to easily get all the advantages of tmux while not having to learn the commands, shortcuts, and footguns

easily: 
- manage sessions/windows instead of dealing with a ton of tabs
- survive disconnects on ssh connections
- run long running jobs

the ui controls use standard nvim controls hjkl for movement etc

it also has the key customizations useful for nvim users right out of the box
